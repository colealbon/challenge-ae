# Code Challenge (AE)
skills test from the Scottsdale area

### installation
* ```git clone https://github.com/colealbon/challenge-ae.git;```  
* ```cd challenge-ae;```  
* ```npm install;```  
* ```cp config/options.js.example config/options.js;```  
* ```npm test;``` or ```mocha --ui=tdd --harmony test``` (optional)
* ```node index;```
